# Задание по Prolog
## Компилятор
Для разработки использовал SWI-prolog v.8.0.3

## Комментарии по реализации задач
### Задание 1
 Файл Task1.pl
 #### Запуск
 	Start(N, R).  где N - Номер нужного эл-та, R - ответ

#### Вспомогательные предикаты
	nechet(X) - предикат для определения нечетное ли число X
	max_nech(X, Y) - предикат для нахождения максимально близкого(в меньшую сторону) нечетного числа к числу X
	get_n(X, N) - предикат нахождения номер текущего нечетного числа X

### Задание 10
 Файл Task10.pl
 #### Запуск
 	start(Set, Idx1, idx2, A). Где Set - это список, в котором нужно поменять местами 
	эл-ты с индексами Idx1 и Idx2, A - ответ
#### Вспомогательные предикаты
	set_by_id(Set, I, Idx, H, A) - Предикат устанавливающий значение эл-та списка Set с индексом Idx  в H, I - текущий индекс эл-та, A - ответ(список)
	get_by_id(Set, I, Idx, A) - Предикат поучающий значение эл-та списка Set с индексом Idx, I - текущий индекс эл-та, A - ответ(значение)

### Задание 11
 Файл Task11.pl
 #### Запуск
 	start(Set, Sdvig, A). Где Set - это список, который нужно циклически сдвинуть на Sdvig эл-тов, A - ответ
#### Вспомогательные предикаты
	len(Set, A) - Предикат вычесления длинны списка Set, A - ответ
	sdvig(Set, Idx, A) - Предикат, осуществляющий сдвиг списка Set влево на Idx 
	эл-тов, A - ответ
	copy_after(Set, I, Idx, A) - Предикат, копирующий все эл-ты Set в A, текущий индекс которых I <= Idx, A - ответ
	copy_before(Set, I, Idx, A) - Предикат, копирующий все эл-ты Set в A, текущий индекс которых I > Idx, A - ответ

### Задание 13
 Файл Task13.pl
 #### Запуск
 	start(Set, A). Где Set - это список строк матрицы, A - ответ
#### Вспомогательные предикаты
